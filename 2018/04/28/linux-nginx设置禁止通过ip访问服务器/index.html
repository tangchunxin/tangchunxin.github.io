linux nginx设置禁止通过ip访问服务器

为了避免别人把未备案的域名解析到自己的服务器ip而导致服务器被断网，需要在nginx上设置禁止通过ip访问服务器，只能通过域名访问。

  • 最关键的一点是，在server的设置里面添加这么一行：
  • Listen 80 default;
