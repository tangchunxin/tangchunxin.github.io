iconv — 字符串按要求的字符编码来转换

说明

string iconv ( string $in_charset , string $out_charset , string $str )
将字符串 str 从 in_charset 转换编码到 out_charset。

参数

in_charset

  • 输入的字符集。

out_charset

  • 输出的字符集。

str

  • 要转换的字符串。
